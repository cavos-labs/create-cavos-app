# Cavos Starter App

This is a minimal Starknet application powered by [Cavos](https://cavos.xyz).

## Getting Started

1. Set your App ID in `.env.local` (already done if you used the CLI).
2. Run the development server:

```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Features

- **Invisible Wallets**: Onboard users via Google/Apple.
- **Gasless**: Transactions are sponsored by default via AVNU.
- **Self-Custodial**: Real Starknet smart accounts under the hood.
